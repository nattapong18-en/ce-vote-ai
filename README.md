# CE AI Tools Voting

This is a dedicated web application created specifically for Computer Engineering students to conduct an official vote for the best AI tools (Claude, GPT, Gemini) for their academic work.

## 📋 Purpose

This platform serves as a centralized system for gathering student preferences. It is designed to ensure a fair and organized voting process, allowing students to compare technical specifications and cast their single vote for the preferred AI tool.

## 🔒 Voting Policy

* **One Student, One Vote:** Each student ID is limited to exactly one vote to ensure data integrity.
* **Authentication:** Voting is strictly restricted to students using their official student ID.

## 🛠 Features

* **In-depth AI Comparison:** A detailed technical breakdown of available AI tools.
* **Secure Voting:** Real-time data collection via Firebase.
* **Live Results:** Real-time progress tracking to ensure transparency.
